### Student Maagement Information System of Bright Brain
### Overview
  Student Management Information System is a web Application developed to manage students,teachers,attendance,payments,academic records efficiently and based on roles for access to using in system tracking by Manager (Admin)

### Features

## Roles Admin
- Management Academic
  years(create ,update,delete)
  class (view,create,update,delete)
  levels (view,create,update,delete)
  subjects (view,create,update,delete)
- Management User
   user (Create,update,delete)
   Role & Permission (Management Role)
   Approve Permission (Teacher)
- Report (Payment ,student attendance,teacher attendance)
- Management System
  setting (chaneg logo,school Name,school picture)
  Backups (Data store in google drive)
  Security (can update password)
  Activity Log (Tracking)


## Roles Staff
- Management Student (add,update,view,search,filters)
- Management Teacher (add,update,view,search,filers)
- Management Fee (add,update,view,search,filers)
- View Attendance (Teacher and Student)
- Student Payment
- View Report (Student,Teacher)
## Roles Teacher
- Class Management (Student List,Input Score,Student Grade)
- Management Attendance (Take Student Attendance,Check Attendace ,    Permission)
- View Profile

## Role Student
- Class Time Table
- Class Result
- Class Attendance
- Payment
- View Profile
